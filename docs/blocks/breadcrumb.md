# Breadcrumb

Navigation trail showing page location in workspace hierarchy. Autoâ€‘generated by Notion.

## Syntax

```markdown
[breadcrumb]
```

## Examples

```markdown
# API Documentation

[breadcrumb]

Welcome to our API reference.
```

## Builder

```python
from notionary.markdown import MarkdownBuilder

markdown = (MarkdownBuilder()
  .breadcrumb()
  .h1("User Guide")
  .paragraph("Documentation content.")
  .build())
```
